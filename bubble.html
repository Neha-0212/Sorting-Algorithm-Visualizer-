Bubble Sort

Bubble Sort

Bubble Sort is a simple sorting algorithm that involves iterating through a list, comparing adjacent elements, and swapping them if they're in the wrong order. This process continues until the list is arranged in the correct order. The name "Bubble Sort" originates from the way smaller elements move upwards (or "bubble up") to their correct positions as the algorithm progresses, as bubbles rising in water.

Algorithm

Step I : Iterate through the list.

Step II : Compare each pair of adjacent elements.

Step III : If the elements are in the wrong order, swap them.

Step IV : Continue iterating through the list until no more swaps are needed

When To Use

When dealing with a small dataset.
When simplicity of implementation is prioritized over efficiency.
When the list is nearly sorted and few passes are needed for sorting.

Time & Space Complex

Case Complexity
Best Case (Ω) O(n)
Average Case (Θ) O(n2)
Worst Case (O) O(n2)
Space O(1)

                    def bubble_sort(arr):
                        n = len(arr)
                        for i in range(n):
                            for j in range(0, n-i-1):
                                if arr[j] > arr[j+1]:
                                    arr[j], arr[j+1] = arr[j+1], arr[j]
                
                    void bubble_sort(int arr[], int n) {
                        int i, j;
                        for (i = 0; i < n-1; i++) {
                            for (j = 0; j < n-i-1; j++) {
                                if (arr[j] > arr[j+1]) {
                                    int temp = arr[j];
                                    arr[j] = arr[j+1];
                                    arr[j+1] = temp;
                                }
                            }
                        }
                    }
