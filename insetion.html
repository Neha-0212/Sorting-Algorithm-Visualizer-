Insertion Sort

Insertion Sort is a sorting algorithm that iterates through the list, moving one element at a time to its correct position within the sorted subarray. It starts with the assumption that the first element is already sorted and then iteratively adds subsequent elements by comparing them with the elements in the sorted portion.

If an element is smaller than the elements in the sorted part, it's shifted to the left until it reaches its correct position. Insertion Sort is efficient for small lists or nearly sorted data due to its simplicity and low overhead .

Its time complexity of O(n^2) makes it less suitable for large datasets where more efficient algorithms like Quick Sort or Merge Sort are preferred.

Algorithm

Step I : Iterate through the list from the second element to the last element.
Step II : For each element, compare it with the elements to its left.
Step III : If the current element is smaller than the left element, swap them and continue comparing with the previous elements until the correct position is found or the beginning of the list is reached.

When To Use

When handling small datasets or mostly sorted lists.
When prioritizing simplicity and ease of implementation over efficiency.
When maintaining stability in sorting order is necessary.

Time & Space Complex

Case Complexity
Best Case (Ω) O(n)
Average Case (Θ) O(n2)
Worst Case (O) O(n2)
Space O(1)

                def insertionSort(arr):
                n = len(arr)
                for i in range(1, n):
                    key = arr[i]
                    j = i - 1
                    # Move elements of arr[0..i-1], that are greater than key, to one position ahead of their current position
                    while j >= 0 and arr[j] > key:
                        arr[j + 1] = arr[j]
                        j -= 1
                    arr[j + 1] = key

                    for (let i = 1; i < n; i++) {
                        let key = arr[i];
                        let j = i - 1;

                        // Move elements of arr[0..i-1], that are greater than key,

                            arr[j + 1] = arr[j];
                            j--;
                        }
                        arr[j + 1] = key;
                    }
                }
