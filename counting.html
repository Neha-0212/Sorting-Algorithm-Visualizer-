Counting Sort

Counting Sort is a sorting algorithm that efficiently sorts elements by tallying the occurrences of each unique value in the input list. It counts how many times each distinct element appears and uses this information to place the elements in sorted order. By iterating through the input list and updating a count array accordingly, Counting Sort avoids the need for comparisons between elements, which contributes to its efficiency.

Once the counts are determined, the algorithm constructs the sorted output array by placing each element in its correct position based on the count array. Counting Sort's linear time complexity of O(n + k), where n is the number of elements and k is the range of values, makes it particularly useful for datasets with a limited range of values.

It may require additional memory space for the count array proportional to the range of input values.

Algorithm

Step I : Determine the range of input values and create a count array initialized with zeros.
Step II : Iterate through the input list, incrementing the count of each element in the count array.
Step III : Modify the count array to accumulate counts, ensuring each element stores the sum of counts up to its index.
Step IV : Create an output array of the same size as the input list.
Step V : Iterate through the input list again, placing each element into its correct sorted position in the output array based on the count array. Copy the sorted elements back into the original input list.

When To Use

When sorting integers within a known and relatively small range efficiently.
When stability in sorting order is crucial.
- When additional memory usage is not a major concern.

Time & Space Complex

Case Complexity
Best Case (Ω) O(n+k)
Average Case (Θ) O(n+k)
Worst Case (O) O(n+k)
Space O(n+k)

                    def countingSort(arr, maximum):
        n = len(arr)
    
        # Initialize count array with 0
        count = [0] * (maximum + 1)
    
        # Store count of each element in count array
        for num in arr:
            count[num] += 1
    
        # Modify count array to store actual position of elements
        for i in range(1, maximum + 1):
            count[i] += count[i - 1]
    
        # Build the output array
        output = [0] * n
        for num in reversed(arr):
            output[count[num] - 1] = num
            count[num] -= 1
    
        # Copy the output array to arr, so that arr now contains sorted elements
        for i in range(n):
            arr[i] = output[i]
    
                
                        
                        // Function to perform counting sort
                        void countingSort(int arr[], int n, int range) {
                            int output[n]; // Output array to store sorted elements
                            int count[range + 1]; // Count array to store count of elements
                        
                            // Initialize count array with 0
                            for (int i = 0; i <= range; ++i) {
                                count[i] = 0;
                            }
                        
                            // Store count of each element in count array
                            for (int i = 0; i < n; ++i) {
                                ++count[arr[i]];
                            }
                        
                            // Modify count array to store actual position of elements
                            for (int i = 1; i <= range; ++i) {
                                count[i] += count[i - 1];
                            }
                        
                            // Build the output array
                            for (int i = 0; i < n; ++i) {
                                output[count[arr[i]] - 1] = arr[i];
                                --count[arr[i]];
                            }
                        
                            // Copy the output array to arr, so that arr now contains sorted elements
                            for (int i = 0; i < n; ++i) {
                                arr[i] = output[i];
                            }
                        }

                    // Function to perform counting sort
                    function countingSort(arr, maximum) {
                        const n = arr.length;
    
                        // Initialize count array with 0
                        const count = Array(maximum + 1).fill(0);
    
                        // Store count of each element in count array
                        for (let i = 0; i < n; i++) {
                            count[arr[i]]++;
                        }
    
                        // Modify count array to store actual position of elements
                        for (let i = 1; i <= maximum; i++) {
                            count[i] += count[i - 1];
                        }
    
                        // Build the output array
                        const output = new Array(n);
                        for (let i = n - 1; i >= 0; i--) {
                            output[count[arr[i]] - 1] = arr[i];
                            count[arr[i]]--;
                        }
    
                        // Copy the output array to arr, so that arr now contains sorted elements
                        for (let i = 0; i < n; i++) {
                            arr[i] = output[i];
                        }
                    }
    
                
                    public class CountingSort {
    
                        // Function to perform counting sort
                        public static void countingSort(int[] arr, int range) {
                            int n = arr.length;
    
                            // Initialize count array with 0
                            int[] count = new int[range + 1];
    
                            // Store count of each element in count array
                            for (int num : arr) {
                                count[num]++;
                            }
    
                            // Modify count array to store actual position of elements
                            for (int i = 1; i <= range; i++) {
                                count[i] += count[i - 1];
                            }
    
                            // Build the output array
                            int[] output = new int[n];
                            for (int i = n - 1; i >= 0; i--) {
                                output[count[arr[i]] - 1] = arr[i];
                                count[arr[i]]--;
                            }
    
                            // Copy the output array to arr, so that arr now contains sorted elements
                            System.arraycopy(output, 0, arr, 0, n);
                        }
