radix Sort

radix Sort

Radix Sort is a unique sorting algorithm that organizes elements based on their individual digits or characters. It processes the input list digit by digit or character by character, distributing elements into buckets according to the value of each digit or character. By iterating through the list in this manner, Radix Sort ensures that elements are sorted based on their entire numeric or lexicographic values.

With a time complexity of O(nk), where n is the number of elements and k is the number of digits or characters in the largest element, Radix Sort is particularly efficient for datasets with a fixed number of digits or characters. It may require additional memory space for storing the buckets, especially when handling a wide range of values.

Algorithm

Step I : Identify the maximum number of digits or characters in the input list.
Step II : Iterate through each digit position from least significant to most significant.
Step III : Distribute elements into buckets based on the value of the digit at the current position.
Step IV : Collect elements from buckets back into a single list.
Step V : Repeat distribution and collection process for each digit position, sorting the elements by their entire numeric or lexicographic values.

When To Use

When sorting integers or strings with fixed lengths efficiently.
When sorting integers or strings with fixed lengths efficiently.
When maintaining stability in sorting order is necessary.

Time & Space Complex

Case Complexity
Best Case (Ω) O(nk)
Average Case (Θ) O(nk)
Worst Case (O) O(nk)
Space O(n+k)

            def countingSort(arr, exp):
                n = len(arr)
                output = [0] * n
                count = [0] * 10

                # Store count of occurrences in count[]
                for i in range(n):
                    index = arr[i] // exp
                    count[index % 10] += 1

                # Change count[i] so that count[i] now contains actual position of this digit in output[]
                for i in range(1, 10):
                    count[i] += count[i - 1]

                # Build the output array
                i = n - 1
                while i >= 0:
                    index = arr[i] // exp
                    output[count[index % 10] - 1] = arr[i]
                    count[index % 10] -= 1
                    i -= 1

                # Copy the output array to arr[], so that arr[] now contains sorted numbers according to current digit
                for i in range(len(arr)):
                    arr[i] = output[i]
