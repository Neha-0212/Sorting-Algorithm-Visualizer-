Bucket Sort

Bucket Sort is a sorting algorithm that divides the input list into a fixed number of buckets, each representing a specific range of values. Elements from the list are then placed into these buckets based on their values. Once all elements are distributed, each bucket is individually sorted.

This can be done using any sorting algorithm suitable for the range of values within each bucket. The sorted elements from all buckets are concatenated to produce the final sorted list. Bucket Sort is particularly effective when the input values are uniformly distributed across a range and when the number of buckets is chosen appropriately.

It may require additional memory space for storing the buckets, and its performance may degrade if the input distribution is skewed or if the number of buckets is insufficient.

Algorithm

Step I : Divide the range of input values into fixed-size buckets.
Step II : Place each element from the input list into the appropriate bucket.
Step III : Sort each bucket individually using a suitable sorting algorithm.
Step V : Concatenate the sorted buckets to form the final sorted list.

When To Use

When dealing with a small dataset.
When simplicity of implementation is prioritized over efficiency.
When the list is nearly sorted and few passes are needed for sorting.

Time & Space Complex

Case Complexity
Best Case (Ω) O(n+k)
Average Case (Θ) O(n+k)
Worst Case (O) O(n2)
Space O(n+k)

                    bucket = []

                    # Create empty buckets
                    for i in range(len(array)):
                        bucket.append([])

                    # Insert elements into their respective buckets
                    for j in array:
                        index_b = int(10 * j)
                        bucket[index_b].append(j)

                    # Sort the elements of each bucket
                    for i in range(len(array)):
                        bucket[i] = sorted(bucket[i])

                    # Get the sorted elements
                    k = 0
                    for i in range(len(array)):
                        for j in range(len(bucket[i])):
                            array[k] = bucket[i][j]
                            k += 1
                    return array

                // Function to perform bucket sort
                function bucketSort(arr) {
                    const n = arr.length;
                
                    // Create empty buckets
                    const buckets = new Array(n);
                    for (let i = 0; i < n; i++) {
                        buckets[i] = [];
                    }
                
                    // Put array elements in different buckets
                    for (let i = 0; i < n; i++) {
                        const bucketIndex = Math.floor(n * arr[i]);
                        buckets[bucketIndex].push(arr[i]);
                    }
                
                    // Sort individual buckets
                    for (let i = 0; i < n; i++) {
                        buckets[i].sort((a, b) => a - b);
                    }
                
                    // Concatenate all buckets into arr[]
                    let index = 0;
                    for (let i = 0; i < n; i++) {
                        for (const value of buckets[i]) {
                            arr[index++] = value;
                        }
                    }
                }
            
                public class BucketSort {

                    // Function to perform bucket sort for integer values
                    public static void bucketSort(int[] arr) {
                        int n = arr.length;

                        // Determine maximum and minimum values in the array
                        int maxVal = Integer.MIN_VALUE;
                        int minVal = Integer.MAX_VALUE;
                        for (int num : arr) {
                            maxVal = Math.max(maxVal, num);
                            minVal = Math.min(minVal, num);
                        }

                        // Calculate bucket size
                        int bucketSize = (maxVal - minVal) / n + 1;

                        // Create empty buckets
                        List[] buckets = new List[n];
                        for (int i = 0; i < n; i++) {
                            buckets[i] = new ArrayList<>();
                        }

                        // Put array elements in different buckets
                        for (int i = 0; i < n; i++) {
                            int bucketIndex = (arr[i] - minVal) / bucketSize;
                            buckets[bucketIndex].add(arr[i]);
                        }

                        // Sort individual buckets
                        for (int i = 0; i < n; i++) {
                            Collections.sort(buckets[i]);
                        }

                        // Concatenate all buckets into arr[]
                        int index = 0;
                        for (int i = 0; i < n; i++) {
                            for (int value : buckets[i]) {
                                arr[index++] = value;
                            }
                        }
                    }
