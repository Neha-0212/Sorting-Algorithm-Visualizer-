Selection Sort

Selection Sort is a sorting algorithm that repeatedly identifies the smallest (or largest) element from the unsorted part of the list and places it at the beginning of the sorted section. It iterates through the list, selecting the minimum (or maximum) element each time and swapping it with the current element at the beginning of the unsorted sublist.

This process gradually builds up the sorted section of the list until all elements are sorted. Despite its simplicity, Selection Sort's time complexity of O(n^2) makes it less efficient compared to other sorting algorithms like Merge Sort or Quick Sort, especially for large datasets.

It remains useful for educational purposes and sorting small lists due to its straightforward implementation.

Algorithm

Step I : Find the minimum (or maximum) element in the unsorted part of the list and swap it with the first unsorted element.
Step II : Move the boundary between the sorted and unsorted parts one element to the right.
Step III : Repeat steps 1 and 2 until the entire list is sorted.

When To Use

When dealing with a small dataset.
When simplicity of implementation is prioritized over efficiency.
When the list is nearly sorted and few passes are needed for sorting.

Time & Space Complex

Case Complexity
Best Case (Ω) O(n2)
Average Case (Θ) O(n2)
Worst Case (O) O(n2)
Space O(1)

                    // Function to perform selection sort
                    void selectionSort(int arr[], int n) {
                        int i, j, min_idx;
                    
                        // Traverse through all array elements
                        for (i = 0; i < n - 1; i++) {
                            // Find the minimum element in the unsorted part of array
                            min_idx = i;
                            for (j = i + 1; j < n; j++) {
                                if (arr[j] < arr[min_idx]) {
                                    min_idx = j;
                                }
                            }
                    
                            // Swap the found minimum element with the first element
                            int temp = arr[min_idx];
                            arr[min_idx] = arr[i];
                            arr[i] = temp;
                        }
                    }
